자바의 장점
- 모든 운영체제에서 실행가능하다. (C언어는 체제별로 짜줘야함)
- 객체지향 프로그래밍(OOP: Object-Oriented Programming)
- 메모리 자동 정리(양날의 검: 빠르게 개발가능하지만 메모리 용량이 작은 기기에서는(iot?) 부적합)
- 풍부한 무료 라이브러리(≒참고자료)
- 자바를 한마디로 표현하자면 재사용이다.
